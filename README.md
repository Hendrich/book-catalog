# Book Catalog App

A simple fullstack demo app to manage a book list using:

- 🖥️ HTML for frontend
- ⚙️ Node.js + Express for backend
- 📂 JSON file as database

## Features

- Add and list books via web
- Store book data in a JSON file
- API endpoint: `/api/books`

## Run Locally

```bash
npm install
npm start
```
